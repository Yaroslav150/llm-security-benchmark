# LLM Security Benchmark

## Overview
This project benchmarks LLM robustness against prompt injection attacks.

## Features
- Dataset of adversarial prompts
- Simulated model evaluation
- Attack Success Rate (ASR)
- CSV logging of results

## Methodology
Each prompt is tested against a model simulation:
- Attack applied
- Response generated
- Success evaluated

## Metrics
- Attack Success Rate (ASR)

## Output
Results are stored in `results.csv`.
